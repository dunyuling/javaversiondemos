##OPENJDK与JDK的区别
    + 授权协议不同
        GPL协议 GRL协议  
        OPENJDK GPL协议
        JDK   GRL协议
        既然JDK有很多源码来自OPENJDK,那么为什么JDK可以以新的GRL协议发行哪？
    + OpenJDk不包含部署功能
        * java web start
        * java 控制面板
    * OpenJDk 源代码不完整(产权原因)
    + 部分源代码使用开源代码替换(产权原因)
        * 字体功能化引擎
        * ...
    * OpenJDK只包含最精简的JDK
    * 不能使用Java商标
